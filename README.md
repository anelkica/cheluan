# 🐢 cheluan
> *chelona (turtle) + lua + moon (MoonSharp) = cheluan*

A C# turtle graphics playground with embedded Lua for controlling the turtle.

Load `.lua` scripts or code them in the application to see changes instantly!

[⟹ See screenshots!](#-screenshots)

## ✨ Features
- Live scripting: write Lua code to draw on the canvas
    - Run or Save (CTRL + S) to see visual changes
- Load and Save `.lua` scripts
- Syntax highlighting
  - Courtesy of AvaloniaEdit and TextMate!
