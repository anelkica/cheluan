> *chelona (turtle) + lua + moon (MoonSharp) = cheluan*

A C# turtle graphics playground with embedded Lua for controlling the turtle.

Load `.lua` scripts or code them in the application to see changes instantly.

[⟹ See screenshots](#-screenshots)

## 🎨 Live scripting
Write Lua code and see visuals on the canvas immediately.

- **Run** code with `Run`, `CTRL+S` or `Auto-run` as you type
- **Create, open** or **save** `.lua` scripts
- **Export** canvas as transparent PNG

## ✨ Features
- Built-in cheatsheet known as `Docs` for methods
- Syntax **highlighting**
  - Courtesy of AvaloniaEdit and TextMate
- Built-in simple **autocomplete**
