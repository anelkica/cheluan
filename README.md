# 🐢 cheluan
> *chelona (turtle) + lua + moon (MoonSharp) = cheluan*

A C# turtle graphics playground with embedded Lua for controlling the turtle.

Load `.lua` scripts for amazing visuals, or code them in the app and see changes immediately!

## ✨ Features
- Live scripting: write Lua code to draw on the canvas, no compiling required
